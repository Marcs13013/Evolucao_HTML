      <"ul"> é usada para criar uma lista não ordenada, enquanto a tag
      <"ol"> é usada para criar uma lista ordenada. Dentro de ambas as tags, você usa a tag
      <"li"> para criar itens de lista individuais.

      Lembre-se de que esses são exemplos simples, e você pode personalizar as listas com estilos CSS e incorporá-las em um layout mais complexo, se necessário.
