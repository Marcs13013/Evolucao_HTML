Evolução Do HTML

HTML (Hypertext Markup Language) é uma linguagem de marcação de hipertexto utilizada para criar e exibir conteúdo em uma página web. A sua evolução começou em 1989, quando Tim Berners-Lee, um cientista da computação britânico que trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear), desenvolveu a ideia da World Wide Web. Essa ideia proporcionou o benéfico ato de qualquer pessoa poder compartilhar conhecimento na web utilizando linguagem HTML. A web foi então construída em cima da própria internet, a rede mundial que liga todos os computadores.
