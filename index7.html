Exemplo Tags de texto


No HTML existem variás tags para inserir texto em seu site, as principais tags são:
<"p">: Esse é usado para criar um paragrágo, uma linha de texto. aqui está um exemplo:

(<'p'> frase teste de exemplo da tag de paragráfo <'/p'>)

frase teste de exemplo da tag de paragráfo



<"/br">: Essa tag é usada para fazer uma quebra de linha em seu texto, é uma substituição da tecla "Enter" do computador. Exemplo:

<"p"> esse é um texto <"/br"> com 2 quebras<"/br"> de linha<"/p">
