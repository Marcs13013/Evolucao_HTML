Exemplo de Tag IMG


A tag <"img"> é usada em HTML para incorporar imagens em uma página web. Ela não tem uma tag de fechamento e é considerada uma "tag vazia" ou "tag autossuficiente". Aqui está a estrutura básica da tag <"img">. Exemplo - Você baixa uma imagem de um gato preto e quer colocar em seu site, para isso usa-se o código:
<"img src="caminho da imagem(GatoPreto.jpg)" alt="texto_descritivo(Gato Preto)" width="200" height="200">

src: Este é o atributo obrigatório que especifica o caminho da imagem a ser exibida. Pode ser um caminho local no servidor ou uma URL externa. Exemplo: (src="imagem.jpg" ou src="https://www.example.com/imagem.jpg").

alt: Este é um atributo de texto alternativo para a imagem. É usado para acessibilidade, permitindo que os leitores de tela descrevam a imagem para usuários com deficiência visual. Também é exibido quando a imagem não pode ser carregada. Exemplo: alt="Uma imagem de gato".

width e height: Esses é um atributo que permite especificar a largura e a altura da imagem em pixels. Isso pode ser útil para controlar o tamanho da imagem exibida na página. Deve ser usado com cuidado, para não distorcer a proporção original da imagem.

title: Este é um atributo opcional que fornece um texto de dica de ferramenta (tooltip) quando o usuário passa o mouse sobre a imagem.

border: Este é um atributo opcional que define a largura da borda ao redor da imagem em pixels.

Código IMG Na Prática



<"img src="GatoPreto.jpg" alt="Gato Preto"" width="200" height="200" title="Gato Preto" border="6">
