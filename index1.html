HTML 1 - História e Conceitos Básicos


A primeira versão do HTML, conhecida como HTML 1.0, foi criada e lançada em 1991. Essa versão incluía tags básicas para formatar o texto, como <"h1"> para títulos, <"p"> para parágrafos e <"a"> para links. Também incluía imagens, embora apenas em preto e branco.
